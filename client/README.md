## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- yarn
  ```sh
  npm install yarn -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/haemie/supertictactoe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
